Structure document html : 
				<!DOCTYPE html> <!-- Définition du type de document -->
				<html>
					<head>
						<meta charset="utf-8" />  <!-- Encodage du document (pour les caractères accentués par exemple	 -->
						<title> Titre page </title>
					</head>
					
					<body>
						Texte.....
					<body/>
				</html>
			
				Afficher "&" en texte : &amp;
				Afficher "<" en texte : &lt;
				Afficher ">" en texte : &gt;
